# Odin Recipes

Welcome to the Odin Recipes repository! This project is a practical implementation of HTML and CSS concepts, developed as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## 📖 Project Overview

The goal of this project is to build a basic recipe website. It demonstrates the ability to link multiple HTML pages, structure content semantically, and apply CSS for visual enhancement.

## 🌟 Features

- **Flexbox Layouts**: Utilizes CSS Flexbox (`display: flex`) for centering elements and managing layout direction.
- **Custom Styling**:
  - **Gradient Headers**: Implements `background-clip: text` for colorful, gradient-filled typography.
  - **Interactive Elements**: Hover effects on links to improve user experience.
  - **Image Styling**: Consistent borders and sizing for recipe images.
- **CSS Demonstrations**: Includes specific classes to test and demonstrate CSS rule order and specificity.

## 🛠️ Technologies Used

- **HTML5**: For the structural framework of the website.
- **CSS3**: For styling, layout design, and visual effects.

## 🚀 How to Run

1. Clone this repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your preferred web browser to explore the recipes.

## 📚 What I Learned

- **The Box Model**: Mastering `box-sizing: border-box`, margins, and padding.
- **Selectors & Specificity**: Understanding how different selectors apply styles and how to override them.
- **Flexbox**: Aligning items effectively within containers.
